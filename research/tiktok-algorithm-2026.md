## 3. Key Facts & Data Points

**What the algorithm is:**
- TikTok's recommendation system decides which videos appear on each user's For You Page (FYP)
- Analyzes thousands of signals per user per session
- Runs on two parallel tracks: FYP discovery and Search — both matter for CRP earnings

**Key 2026 algorithm signals (industry consensus from Buffer, Sprout Social, OpusClip, multiple sources):**
1. Completion rate and watch time — #1 signal, estimated 40-50% of weight [UNVERIFIED — percentage is industry estimate]
2. Shares and saves — now outweigh likes
3. Comments — quality (length/depth) matters more than count [UNVERIFIED]
4. Rewatch rate — loop triggers a second count, boosts algorithm signal
5. Search value — videos that answer specific queries get boosted by search algorithm separately

**2026 specific changes (vs. 2024-2025):**
- **Follower-first testing:** New videos shown primarily to existing followers first (rolled out late 2025). Follower engagement determines whether video gets pushed to non-followers. This is a significant change — means strong follower relationships now matter for reach, not just vanity count. [Multiple industry sources; not official TikTok documentation]
- **Completion rate bar raised:** Virality threshold moved from ~50% to ~70% completion rate [UNVERIFIED — consistent industry reporting]
- **Search integration deepened:** 49% of US consumers use TikTok as a search engine (Adobe, January 2026); TikTok scans spoken keywords, on-screen text, and captions for search ranking
- **Oracle deal context:** In January 2026, TikTok signed agreement to divest 45% of US operations to Oracle/Silver Lake/MGX investor group; algorithm to be licensed from ByteDance and retrained on US user data [factual, from news coverage]

**How the algorithm processes a new video:**
1. Video released to small initial cohort (existing followers + similar-interest users)
2. Algorithm measures completion rate, saves, shares, rewatches within first 1-3 hours
3. If metrics pass threshold, video enters next distribution wave (larger audience)
4. Process repeats — waves can continue for days or weeks for high-performing content
5. Videos can go viral weeks after posting if algorithm keeps distributing

**Search algorithm layer (unique angle for TCP):**
- TikTok scans 3 layers: spoken keywords (transcribed audio), on-screen text, captions/hashtags
- Words spoken in first 5 seconds carry strongest search signal
- Adobe January 2026 study: 49% of US consumers have used TikTok as search engine; among Gen Z, 64% use TikTok for search
- This means a video optimized for search earns two traffic sources: FYP + search results
- Search views are also counted as qualified views for CRP [UNVERIFIED — needs verification; community assumes yes]

**What does NOT directly affect the algorithm:**
- Follower count (confirmed by TikTok newsroom) — a 200 follower account can beat a 2M account
- Posting frequency alone (without quality)
- Account age

**How algorithm affects CRP earnings specifically:**
- FYP virality and Search virality have different CRP earning profiles
- FYP viral video: high views, often global distribution → can dilute RPM if non-US/non-tier-1 audience
- Search viral video: lower volume but highly targeted, higher intent, often better RPM [community insight, not TikTok official]
- Documenting this distinction is a unique TCP insight from Phase 2 research
