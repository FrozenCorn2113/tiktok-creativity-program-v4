## Core Web Vitals basics
- Use appropriately sized images and avoid oversized assets.
- Prefer system fonts or preload the primary font.
- Keep CSS minimal and avoid heavy animations.
- Defer non-critical scripts where possible.

## Accessibility basics
- Ensure visible focus states for keyboard users.
- Provide descriptive labels for inputs and buttons.
- Use sufficient color contrast for text and controls.
- Avoid relying on color alone to convey meaning.

## Content trust checks
- Include independent and affiliate disclosures.
- Avoid inflated or unverifiable claims.
- Keep CTAs clear and non-coercive.
