## CT1: How-To Tip

**Pillar:** Actionable tips
**Best for:** Instagram carousel, TikTok video, Pinterest infographic, X thread

### Structure

```
HOOK: [One-line question or bold claim about the tip topic]

STEPS:
1. [Step with specific action]
2. [Step with specific action]
3. [Step with specific action]
4. [Optional: Step with specific action]
5. [Optional: Step with specific action]

TAKEAWAY: [One sentence summarizing the outcome of following these steps]

CTA: [Platform-appropriate call to action]
```

### Platform Versions

**Instagram (carousel):**
- Slide 1: Hook question as headline
- Slides 2-5: One step per slide with supporting detail
- Final slide: Takeaway + "Save this for later" CTA
- Caption: 125-150 chars summarizing the tip, keyword-rich
- Hashtags: Use Set A-E rotation (3-5 tags, 1-2-2 rule)

**TikTok (video concept):**
- First 3 seconds: Hook question on screen text
- Caption: Hook in first 100 chars, keyword-optimized
- Hashtags: 3-5 from TikTok rotation sets
- Video must be 60+ seconds (Creator Rewards eligibility)

**X (tweet or thread):**
- Single tweet: Hook + compressed steps (200-250 chars)
- Thread version: Hook tweet, then one tweet per step
- 1-2 hashtags max

**Pinterest (infographic pin):**
- Keyword-rich title: "How to [Topic] on TikTok in [Year]"
- 2:3 ratio design (1000x1500px)
- SEO description with long-tail phrases
- 2-5 hashtags in description

**Facebook (text post):**
- 100-250 word expanded version with context
- No hashtags or 1-2 max

**Reddit (comment format):**
- Standalone value, no links, no promotional language
- Written as helpful advice from a fellow creator

### Example Fill

```
HOOK: Your TikTok RPM is probably lower than it should be. Here's why.

1. Check your audience location in Analytics > Followers. US/UK/Canada viewers pay 5-10x more than other regions.
2. Post when your target geography is awake, not when you're awake. Use Analytics > Follower Activity.
3. Stop posting under 60 seconds. Videos under 1 minute don't qualify for Creator Rewards at all.
4. Drop trending sounds if they don't match your niche. Relevance beats virality for RPM.

TAKEAWAY: RPM comes from where your viewers are and how long they watch, not how many views you get.

CTA: Save this and check your analytics tonight.
```

## CT2: Myth vs Fact

**Pillar:** Debunking misconceptions
**Best for:** TikTok video, Instagram carousel, X thread, Facebook post

### Structure

```
MYTH: "[Common misconception stated confidently]"

ACTUALLY: [The real answer, stated plainly with specifics]

PROOF: [Data point, official source, or observable evidence]

CTA: [Platform-appropriate call to action]
```

### Platform Versions

**Instagram (carousel):**
- Slide 1: "MYTH:" in bold with the misconception
- Slide 2: "ACTUALLY:" with the correction
- Slide 3: Proof point with source
- Slide 4: CTA
- Caption: 125-150 chars with keyword
- Hashtags: 3-5, rotation set

**TikTok:**
- Open with the myth stated seriously, then correct it
- Text on screen: MYTH / FACT labels
- Caption: First 100 chars = the myth (hooks curiosity)
- 60+ seconds, 3-5 hashtags

**X:**
- Single tweet: "Myth: [X]. Reality: [Y]. [Proof point]."
- 1-2 hashtags

**Facebook:**
- 100-250 words expanding on the correction
- Include the source for the proof point

**Reddit:**
- Comment format only: answer someone's question with the correction
- Include proof point, no TCP mention

### Example Fill

```
MYTH: "You need 10,000 followers to make money on TikTok."

ACTUALLY: The Creator Rewards Program requires 10,000 followers to join. But follower count after that point has zero effect on how much you earn. A 12K account and a 500K account with the same views and RPM make the same money.

PROOF: TikTok's Creator Rewards FAQ states payouts are based on qualified views, not follower count.

CTA: Stop chasing followers. Start optimizing your RPM.
```

## CT3: Tool Review

**Pillar:** Creator tools and resources
**Best for:** Instagram, Pinterest, X, Facebook

### Structure

```
TOOL: [Tool name]
WHAT IT DOES: [One sentence, plain language]
WHO IT'S FOR: [Specific creator type or use case]
STANDOUT FEATURE: [One thing that makes it worth using]
VERDICT: [Honest 1-2 sentence assessment]
LINK: [Affiliate link placeholder or direct URL]
```

### Platform Versions

**Instagram:**
- Single image or carousel showing tool in use
- Caption: Tool name + verdict + who should use it (125-150 chars)
- Hashtags: 3-5, rotation set

**X:**
- Tweet: "[Tool name] does [thing] for [who]. Worth it if [condition]. Not worth it if [condition]."
- 1-2 hashtags

**Pinterest:**
- Title: "Best [Tool Category] for TikTok Creators [Year]"
- SEO description with long-tail keywords
- 2:3 ratio pin

**Facebook:**
- 100-250 words with honest pros/cons
- 1-2 hashtags max

**Reddit:**
- Only share if someone asks for tool recommendations
- Never lead with affiliate links
- Describe the tool honestly, mention drawbacks

### Example Fill

```
TOOL: CapCut
WHAT IT DOES: Free video editor with auto-captions, templates, and effects built for short-form video.
WHO IT'S FOR: TikTok creators who want polished edits without paying for Premiere Pro.
STANDOUT FEATURE: Auto-captions are surprisingly accurate, and you can style them to match your brand in seconds.
VERDICT: Best free editor for TikTok. The desktop version handles longer edits better than mobile. Export quality is solid.
LINK: [affiliate link or capcut.com]
```

## CT4: Stat/Data

**Pillar:** Platform stats and data
**Best for:** Instagram, X, TikTok, LinkedIn

### Structure

```
STAT: [Bold data point with source]

CONTEXT: [1-2 sentences explaining what this number means]

WHAT THIS MEANS FOR YOU: [Actionable takeaway for a TikTok creator]

CTA: [Platform-appropriate call to action]

SOURCE: [Where this came from -- mark [UNVERIFIED] if not confirmed]
```

### Platform Versions

**Instagram:**
- Single image with the stat as large text
- Caption: Context + actionable takeaway (125-150 chars)
- Hashtags: 3-5, rotation set

**X:**
- Tweet: "[Stat]. [One-line takeaway]. [Source]."
- 1-2 hashtags

**TikTok:**
- Stat on screen, explain context in voiceover
- 60+ seconds, 3-5 hashtags

**Facebook:**
- 100-250 words expanding on the implications

**Reddit:**
- Share in relevant discussion threads as supporting evidence

```
STAT: TikTok creators in the Creativity Program report RPMs between $0.40 and $1.00 per 1,000 qualified views.

CONTEXT: That's significantly higher than the old Creator Fund, which paid $0.02-$0.04 per 1,000 views. The gap between the two programs is not small.

WHAT THIS MEANS FOR YOU: If you're still on the old Creator Fund, switching to Creator Rewards is the single biggest pay raise available to you right now.

CTA: Check your eligibility in your TikTok settings today.

SOURCE: Creator-reported data across multiple forums and communities. RPM varies by niche and region. [UNVERIFIED: exact range based on community reports, not official TikTok data]
```

## CT5: News Commentary

**Pillar:** Platform updates and industry news
**Best for:** X, TikTok, Instagram, Facebook

### Structure

```
NEWS: [What happened, one sentence]

TCP TAKE: [Our perspective on what this means for creators]

WHAT YOU SHOULD DO: [Specific action step]

CTA: [Platform-appropriate call to action]
```

### Platform Versions

**X:**
- Tweet: "[News hook]. [TCP take in one line]. [Action step]."
- Best platform for news commentary, post within hours of news breaking
- 1-2 hashtags

**TikTok:**
- Quick-react video, can be shorter format
- Text on screen: the headline
- 60+ seconds for Creator Rewards eligibility

**Instagram:**
- Story or Reel, not feed post (news is time-sensitive)
- If feed post: carousel with context

**Facebook:**
- 100-250 word take with more context than other platforms

### Example Fill

```
NEWS: Meta launched Creator Fast Track on March 18, 2026, letting creators cross-post TikTok videos to Facebook and Instagram Reels for bonus payouts.

TCP TAKE: This is Meta paying you to do what you should already be doing: repurposing content. The difference is now there's a direct financial incentive for the first 90 days.

WHAT YOU SHOULD DO: Sign up for Creator Fast Track, set up cross-posting from TikTok to Reels, and track whether the bonus payouts justify the extra step. Don't change your content strategy for it.

CTA: Already cross-posting? Reply with your results so far.
```

## CT6: Engagement Question / Poll

**Pillar:** Community engagement
**Best for:** X, Instagram Stories, Facebook, TikTok

### Structure

```
QUESTION: [Specific, opinionated question that creators have strong feelings about]

OPTIONS (if poll):
A. [Option]
B. [Option]
C. [Option -- optional]
D. [Option -- optional]

FOLLOW-UP: [Comment prompt to keep the conversation going]
```

### Platform Versions

**X:**
- Tweet with poll feature (2-4 options)
- Follow up with a reply sharing TCP's perspective after votes come in
- 1-2 hashtags

**Instagram Stories:**
- Poll sticker or question sticker
- Follow up next day with results + commentary

**Facebook:**
- Text post with poll or open-ended question
- 1-2 hashtags max

**TikTok:**
- Question on screen, ask for answers in comments
- 60+ seconds, use the question as your hook

**Reddit:**
- Ask the question genuinely in relevant subreddit
- No TCP branding, just honest curiosity

### Example Fill

```
QUESTION: What's holding you back from hitting 10K followers on TikTok?

OPTIONS:
A. My niche is too small
B. The algorithm isn't showing my content
C. I don't post consistently enough
D. I have no idea what I'm doing wrong

FOLLOW-UP: If you picked B, what does your posting schedule actually look like? Curious how many of you are posting 3+ times per week.
```

## CT7: Thread / Carousel (Deep Dive)

**Pillar:** In-depth educational content
**Best for:** X thread, Instagram carousel, Pinterest, LinkedIn

### Structure

```
HOOK: [Bold claim or question that promises value]

POINT 1: [Subheading + 2-3 sentences of explanation]
POINT 2: [Subheading + 2-3 sentences of explanation]
POINT 3: [Subheading + 2-3 sentences of explanation]
POINT 4: [Subheading + 2-3 sentences of explanation]
POINT 5: [Optional: Subheading + 2-3 sentences]

SUMMARY: [One-line recap of the core takeaway]

CTA: [Save, share, or follow prompt]
```

### Platform Versions

**X (thread):**
- Tweet 1: Hook (this is the only tweet most people see, make it count)
- Tweets 2-5: One point per tweet, self-contained
- Final tweet: Summary + CTA
- 1-2 hashtags on first tweet only

**Instagram (carousel):**
- Slide 1: Hook as headline
- Slides 2-6: One point per slide
- Final slide: Summary + CTA
- Caption: 125-150 chars, keyword-rich
- Hashtags: 3-5, rotation set

**Pinterest:**
- Long-form infographic pin (1000x2100px or similar)
- All points on one pin with clear visual hierarchy
- SEO title and description

**LinkedIn:**
- Text post format, 300-600 words
- Hook in first line (before "see more" fold)
- No hashtags or 2-3 max

### Example Fill

```
HOOK: 5 things I wish I knew before joining the TikTok Creativity Program.

POINT 1: "Qualified views" and "views" are not the same thing.
Only views from users who watch your video for a meaningful duration count toward your payout. A video with 1M views might only have 400K qualified views.

POINT 2: Your RPM depends more on WHERE your viewers are than WHAT you post.
US, UK, Canada, and Australia viewers generate the highest RPMs. If most of your audience is outside those regions, your per-view earnings will be lower.

POINT 3: Videos under 60 seconds don't earn anything.
This catches a lot of creators off guard. The minimum video length for Creator Rewards is 1 minute. Shorter videos are ineligible no matter how many views they get.

POINT 4: Originality is enforced, and they check.
Reposted content, compilations, and heavily templated videos get flagged. The program uses originality checks, and repeated violations can get you removed.

POINT 5: The old Creator Fund and Creator Rewards are different programs.
If you were on the Creator Fund, you need to switch manually. The Creator Rewards Program pays significantly more per view, but you have to opt in.

SUMMARY: Know the rules before you join. The program pays well, but only if your content qualifies.

CTA: Save this so you don't learn these the hard way.
```

## Usage Notes

- **Content ratio (weeks 1-4):** 100% value content. No promotional posts. No TCP links in social captions.
- **Content ratio (week 5+):** 4-1-1 rule. 4 value posts, 1 curated/shared post, 1 promotional post.
- **Hashtag rotation:** Use the sets in HASHTAG_SETS.md. Rotate through A-E to avoid repetition.
- **Reddit:** Never use these templates as direct posts. Adapt the information into comment-style responses. See REDDIT_GUIDE.md.
- **Affiliate links:** Not included in social posts during weeks 1-4. After that, tool reviews (CT3) and thread deep dives (CT7) can include links where natural.
- **All stats must be sourced or marked [UNVERIFIED].** No fictional data points. No invented creator case studies.
