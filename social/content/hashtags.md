## Instagram (3-5 tags per post, 1-2-2 rule)

The 1-2-2 rule: 1 broad tag + 2 niche tags + 2 community tags. Keeps reach wide while targeting the right audience.

| Set | Broad (1) | Niche (2) | Community (2) |
|---|---|---|---|
| A | #TikTokTips | #TikTokMonetization #CreatorRewardsProgram | #ContentCreator #TikTokGrowth |
| B | #CreatorEconomy | #TikTokRPM #TikTokCreatorFund | #TikTokCreator #SocialMediaTips |
| C | #TikTok2026 | #CreatorRewards #TikTokEarnings | #ContentCreatorTips #CreatorLife |
| D | #SocialMediaGrowth | #TikTokStrategy #TikTokAnalytics | #TikTokCommunity #CreatorsOfTikTok |
| E | #MakeMoneyOnline | #TikTokCreatorRewards #VideoMonetization | #TikTokEducation #DigitalCreator |

**Usage notes:**
- Instagram captions should be keyword-rich. Hashtags supplement keywords, not replace them.
- Keyword-rich captions drive approximately 30% more reach than hashtag-heavy captions (per Christopher's research).
- Place hashtags at the end of the caption, not buried in the middle.

## TikTok (3-5 tags per post)

TikTok's algorithm classifies content through video and audio analysis, not primarily through hashtags. Tags are for discoverability, not algorithmic classification. Keep them relevant and minimal.

| Set | Tags |
|---|---|
| A | #TikTokTips #CreatorRewards #TikTokMonetization #ContentCreator #TikTokGrowth |
| B | #TikTokCreator #RPM #TikTokEarnings #CreatorEconomy #TikTok2026 |
| C | #TikTokStrategy #TikTokCreatorRewards #MakeMoneyOnTikTok #ContentCreatorTips #SocialMediaTips |
| D | #TikTokHelp #CreatorRewardsProgram #TikTokAnalytics #VideoMonetization #CreatorTips |
| E | #TikTokEducation #TikTokMoney #CreatorFund #TikTokGrowthTips #ContentStrategy |

**Usage notes:**
- 3-5 tags is the sweet spot. More than 5 adds clutter without measurable benefit.
- First 100 characters of the caption matter more than hashtags for discoverability.
- Avoid banned or shadow-restricted tags. Check each tag has active content before using.

## Pinterest (2-5 tags in description, SEO-first)

Pinterest is a search engine. Hashtags serve as keyword signals, not social discovery. Long-tail phrases outperform single-word tags.

| Set | Tags |
|---|---|
| A | #TikTokCreatorRewards #HowToMakeMoneyOnTikTok #TikTokTips2026 |
| B | #TikTokMonetization #CreatorRewardsProgram #TikTokEarningsGuide |

| D | #BestVideoEditingApps #TikTokForBeginners #TikTokGrowthStrategy |
| E | #TikTokCreatorTips #MakeMoneyCreatingContent #SocialMediaMonetization |

**Usage notes:**
- SEO description matters more than hashtags on Pinterest. Write 2-3 sentences with natural long-tail keywords before adding tags.
- Target keywords: "TikTok Creator Rewards Program," "how to make money on TikTok 2026," "TikTok RPM tips," "best tools for TikTok creators."
- Use full phrases, not abbreviations.

## X (1-2 tags max)

X penalizes hashtag-heavy tweets. Keep it to 1-2 tags max, or skip them entirely if the tweet reads better without them.

| Set | Tags |
|---|---|
| A | #TikTokTips #CreatorEconomy |
| B | #TikTokMonetization #ContentCreator |
| C | #CreatorRewards #TikTok |
| D | #TikTokStrategy #SocialMedia |
| E | #TikTokCreator #CreatorRewardsProgram |

**Usage notes:**
- Tweets without hashtags can perform equally well if the content is strong. Use tags only when they add discoverability.
- Never use more than 2. Tweets with 3+ hashtags see measurably lower engagement.
- For threads, only tag the first tweet.

## Facebook (0-2 tags)

Facebook's algorithm does not rely on hashtags for content distribution. Use 0-2 at most, or skip entirely.

| Set | Tags |
|---|---|
| A | #TikTokTips |
| B | #CreatorRewards |
| C | #TikTokMonetization |
| D | #ContentCreator |
| E | No hashtags |

**Usage notes:**
- Most Facebook posts perform fine without any hashtags.
- If using one, make it the most relevant single tag for the post topic.
- Never use more than 2.
