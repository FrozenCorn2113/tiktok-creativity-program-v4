# Project Intent

- **What the site is:** A content hub for TikTok creators covering the Creator Rewards Program — guides, interactive earnings calculators (RPM, follower income), niche-specific advice, and a downloadable resource library.

- **How users convert:** Email capture via ConvertKit (inline forms, exit-intent popups, newsletter waitlist). Secondary conversions through affiliate links (Amazon gear, Canva, Fiverr, Mavely) and a future premium newsletter/paid products.

- **Payments:** No direct payment processing currently. Revenue model is affiliate commissions + a premium newsletter (not yet launched — waitlist only). No Stripe/PayPal integrated.

- **Auth:** No user authentication. The site is fully public; no login, accounts, or gated content beyond the email list.

- **"Done" looks like:** SEO-indexed Next.js site live on Vercel, all MDX guides rendering, calculators functional, ConvertKit email capture working, affiliate redirects resolving, GA4 tracking active.
